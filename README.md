## 贡献须知

本书目前仍在创作中，如果您想参与到本书创作，欢迎提PR。本书目录结构如下：

| 目录或文件       | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| src             | 文档目录，在此目录下对应的章节文件夹下修改/创建Markdown文档       |
| docs            | 打包后的网站代码目录，使用         |
| src/SUMMARY.md  | 本书目录，要修改目录请参考本文档中前面章节。                     |
| src/images      | 本书引用的图片目录                                             |

在根目录下打开命令行，运行下列命令：
```
gitbook build

gitbook serve
```
